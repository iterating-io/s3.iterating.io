# Repository Template

Use this repository as a starting point for new projects.

## Quick start

1. Click **Use this template** on GitHub.
2. Rename the repository and update this README.
3. Replace placeholder sections/files with project-specific content.

## What to customize

- Project description
- Setup instructions
- Development workflow
- Testing instructions
- License and contribution details

## Development

Add your project tooling and update this section with concrete commands, for example:

```bash
# install dependencies
# run tests
# run lint
```
